Privacy Policy for GhostTab AI

Effective Date: September 9, 2025

GhostTab AI respects your privacy. This extension is designed with user control in mind and does not collect, sell, or share your personal information.

API Key Storage

  • Your API key is stored locally in your browser’s extension storage (chrome.storage.local).
  • We never transmit your API key to third parties other than the backend API endpoint you configure.
  • You may clear or update the key at any time via the Options page.

Data Processing

  • GhostTab AI only communicates with the API base URL you provide.
  • No analytics, advertising, or telemetry frameworks are used.
  • We do not log, store, or share your browsing data.

Security

  • The extension is built on the Chrome MV3 security model and avoids inline scripts.
  • We recommend using secure HTTPS API endpoints whenever possible.
  • Your configuration and key remain private to your browser unless you opt to sync them manually.

Your Control

  • You can clear your stored key at any time in the Options page.
  • Uninstalling the extension removes all locally stored data.
